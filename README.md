# AI Desktop Assistant

An AI-powered desktop assistant built with Python and PyQt6.

## Features

- AI Chat (Gemini)
- Voice Commands
- Text-to-Speech
- Application Control
- Weather Search
- SQLite Chat History
- Modern Dashboard UI
- File Management

## Technologies

- Python
- PyQt6
- Gemini API
- SQLite
- SpeechRecognition
- PyAutoGUI

## Future Scope

- Face Recognition
- Email Automation
- WhatsApp Automation
- Screen Analysis
- AI Agents
